Privacy Policy

1. Information We Collect

MotoMachi collects limited personal information such as name and email address.

The email address is used for:

  • Account login and authentication
  • Account verification
  • Processing account deletion requests

User-generated data may include:

  • Vehicle logs
  • Ride journals
  • Fuel entries
  • Maintenance records

This data is stored securely in a cloud database associated with the user account.

2. How We Use Data

The collected information is used only to provide core application functionality, including:

  • Recording vehicle and ride information
  • Tracking fuel usage and maintenance history
  • Managing ride journals
  • Maintaining user accounts

MotoMachi does not sell or share personal data with third parties.

3. In-App Purchases

MotoMachi may offer premium features through Google Play Billing.

Payments are processed by Google Play and are subject to Google’s privacy policies.

4. Data Storage

User data is stored securely in a cloud database.

Access to this data is restricted to the user account associated with it.

We apply reasonable technical measures to protect user data.

5. Data Deletion

Users can delete their data by:

  • Removing individual entries inside the app
  • Deleting their user profile from the app
  • Requesting account deletion via email

When a user profile is deleted, all associated cloud data is permanently removed.

Uninstalling the application alone does not delete cloud-stored data.

6. Children's Privacy

MotoMachi is not intended for children under the age of 13.

7. Changes to This Policy

This privacy policy may be updated from time to time.

Updates will be published on this page.
